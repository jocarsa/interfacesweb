Formatos de mapas de bits:
  PNG
    puede transparencia
    tiene compresión sin perdida
    max 48 bits de color (normal 24 bits de color)

  JPeG
    Joint Photograph Experts Group
    no tiene transparencia
    tiene compresión variable con pérdida
    max 24 bits = 16.7 Millones de colores
    
  WebP
  	Formato de pixeles
    no tiene transparencia
    Tiene compresión mejor que el jpg
    Actualmente tiene poco soporte en aplicaciones (PS)

Formatos de imágenes vectoriales
	SVG
  	No tienen pixeles
    Tienen vectores
    Resolución infinita
    Pero mal soporte para paletas de colores
    Buenos para logos, malos para fotos
